Crypto Lookup Chrome Extension
Information

This Google Chrome extension aims to help bitcoiners interact with bitcoin-related content on websites by allowing for easier browsing and previewing of addresses, transactions, and block found on websites.

However, it was not free to develop. Please consider donating to the project at 1MF7hKShzq2iSV9ZZ9hEx6ATnHQpFtM7cF (you can use the extension to send to this address). If you have a comment, question, or suggestion, send us an email at btcspry@bitforwarder.com, or open an issue on our GitHub repo.

While the project works very well, we're always looking for developers and users to view the code and make suggestions on our GitHub project! Thanks for using CryptoLookup!
